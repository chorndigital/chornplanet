# Commerce Growth Workflow

Use this workflow when ChornPlanet work is intended to convert content traffic into product interest, marketplace clicks, or future owned-commerce revenue.

## Goal

Connect ChornPlanet content to commerce in a premium, trust-building way.

```text
content -> traffic -> trust -> product story -> marketplace link -> conversion signal -> analytics insight
```

## Starting wedge

Start with:

- lady outfit
- gentleman outfit
- civilization clothing
- premium visual posts
- TikTok-first product links

## Future expansion

- grocery
- online supermarket
- Smart Food meals
- premium lifestyle products
- luxury project products/services
- owned commerce

## Process

1. Define product or product category.
2. Define target audience and platform.
3. Create content/story angle.
4. Create product page or marketplace copy draft.
5. Add product-linking plan.
6. Add analytics tracking idea.
7. Review before publishing.

## Marketplace rollout

Priority:

1. TikTok
2. Shopee
3. Lazada
4. Amazon
5. Owned commerce

## Output artifacts

- product story draft
- marketplace copy draft
- post caption draft
- TikTok script draft
- product-link mapping
- conversion tracking note
- growth experiment note

## Approval rule

Drafting is allowed.

Publishing or changing live marketplace/product data requires explicit approval.
