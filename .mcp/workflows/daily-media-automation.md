# Daily Media Automation Workflow

Use this workflow for future daily news, entertainment, technology, and media follow-up automation.

## Goal

Create a repeatable AI-assisted content engine that supports daily traffic growth:

```text
10K -> 100K -> 500K -> 1M daily traffic
```

## Draft-first pipeline

```text
AI search daily content
  ↓
source review and safety check
  ↓
digest / summary
  ↓
ChornPlanet editorial angle
  ↓
image/video prompt draft
  ↓
post draft
  ↓
review / approval
  ↓
publish
  ↓
analytics review
```

## Categories

Initial categories:

- news
- entertainment
- technology
- outfit/clothing
- Smart Food
- luxury project
- commerce campaign

## Rules

- Keep source tracking for factual content.
- Do not copy full source articles.
- Do not generate misleading news images.
- Do not auto-publish externally without approval.
- Do not mix factual reporting and creative interpretation without clear labeling.

## Output artifacts

Each daily media task should produce one or more of:

- source list
- digest draft
- post draft
- image prompt draft
- video prompt draft
- publishing checklist
- analytics follow-up note
