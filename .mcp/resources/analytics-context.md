# Analytics Context

ChornPlanet should integrate analytics with AI-assisted growth insight.

## Analytics direction

Initial analytics direction:

- Google Analytics integration
- traffic monitoring
- content performance monitoring
- commerce click tracking
- product link tracking
- funnel analysis
- daily/weekly AI-assisted growth summaries

## Growth targets

Analytics should support the traffic path:

```text
10K daily -> 100K -> 500K -> 1M
```

## AI assistant use cases

Future AI assistant workflows may answer:

- Which content categories drive traffic?
- Which outfit posts perform best?
- Which zones perform best?
- Which posts generate commerce clicks?
- Which traffic sources are growing?
- Which pages need SEO improvement?
- Which products should be promoted again?
- Which daily posting strategy should change?

## Data principle

Do not invent metrics.

If analytics data is unavailable, agents should state the gap and recommend instrumentation.

## Privacy and safety

- Prefer aggregated analytics.
- Do not expose personal user data.
- Do not change production analytics configuration without approval.
- Do not connect external analytics credentials without approval.
